1) Briefly describe the purpose of Github.
	Github is a repository hosting service. It allows multiple users to edit a working repository and store the changes to the host.
2) What does a .ignore file do?
	.getIgnore excludes files that get updated to the repository. 
3) What is the purpose of branching?
	Branching leaves the main branch in a working state. This is good for baselining and organization with revision control. 
4) What does the revert command do?
	Reverts to a previous commit. 
5) What does cloning do?
	Clone creates a copy of the local repository to a specified directory. It is still linked to the original repository.
