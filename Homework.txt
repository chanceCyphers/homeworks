1) Briefly describe the purpose of Github.
	To have a centralized storage of a project that multiple people can contribute to

2) What does a .ignore file do?
	Tells Github which files/extensions you want to ignore so you aren’t having to constantly commit things like cache files.

3) What is the purpose of branching?
	To create a clone of a certain branch that you can work on and make changes to, then when you are ready pull back into the main branch without interrupting or breaking working code.

4) What does the revert command do?
	Goes back to a previous commit in a project.

5) What does cloning do?
	Completely clones the full repository to your machine.
